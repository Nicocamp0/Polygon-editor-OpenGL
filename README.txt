# Interactive 2D Polygon Editor

Interactive application written in **C and OpenGL** allowing users to create, edit and fill polygons.

## Features

- Interactive polygon creation with mouse input
- Vertex and edge manipulation
- Scan-line polygon filling algorithm
- Real-time rendering using OpenGL

## Technologies

- C
- OpenGL
- GLUT

## Concepts explored

- Computational geometry
- Scan-line rasterization
- Interactive graphics programming
